# e-Shoku Frontend

オンライン食事会の支援ツール「e-Shoku」のフロントエンド開発リポジトリです。  
[Next.js](https://nextjs.org/)を用いています。

## システム概要
バックエンドは、Django Rest Framework (Python)を用いています。[（リポジトリ）](https://github.com/ryoheinan/e-shoku-api)  
認証にはAuth0を用いています。
